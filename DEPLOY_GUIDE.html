Deploy Butter to GitHub Pages

3 steps · 5 minutes · completely free hosting

1

Download Your Files

Get the 3 files from Claude's output

Download these 3 files from Claude and put them all in one folder on your computer (e.g. a folder called butter-app):

📄butter-pwa.html→ rename to index.html
📋butter-manifest.json→ rename to manifest.json
⚙️butter-sw.js→ rename to sw.js
⚠️ Important: rename butter-pwa.html to exactly index.html — GitHub Pages needs this as the entry point.
2

Create a GitHub Repo

Free account at github.com if you don't have one

1. Go to github.com/new
2. Name the repo: butter-app
3. Set it to Public
4. Click Create repository

💡 No GitHub? Sign up free at github.com — takes 2 minutes.
3

Upload & Enable Pages

Push files and turn on GitHub Pages

Option A — Command Line (fastest, if you have Git installed):

Terminal / Command Prompt
# Navigate into your butter-app folder cd butter-app # Initialize git and push to GitHub git init git add . git commit -m "🧈 Butter loyalty app — investor demo" git branch -M main git remote add origin https://github.com/YOUR-USERNAME/butter-app.git git push -u origin main

Option B — No coding needed (drag & drop in browser):

Browser Steps
1. Open your new repo on github.com 2. Click "uploading an existing file" 3. Drag all 3 files (index.html, manifest.json, sw.js) 4. Click "Commit changes"

Then enable GitHub Pages:

GitHub Settings
Repo → Settings → Pages → Source: "Deploy from a branch" → Branch: main / Folder: / (root) → Click Save
⏱️ Wait ~60 seconds after saving — GitHub will build and deploy your site automatically.
